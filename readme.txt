AEM Tag library
===============

This package provides several tags that can help developers build AEM components quickly.


Installation:
------------

1. Start AEM instance on your computer.

2. Run this command from the root directory to install the library on AEM.
    
	$  mvn clean install -P autoInstallPackage


Demo & Exercise:
---------------

1. Create a new page with the template named 'AEM Tag Library Demo'.
	
2. Edit the demo component named 'Billing Info' found on the page.

3. Refer the file named 'aemTagLibrary.jsp' for the full list of supported tags. Try all the tags.
